Description of the sketch
This sketch draws a line on a canvas that changes direction at a random point. The interation with the user comes with the ability to change certain attributes of the sketch. The first two involve changing the colours of both the background canvas and the line in the foreground. There are various colours that can be selected. The next select menu is the thickness at which the line is draw, with options being from 1 (very thin) to 10 (very thick). Next is the speed at which the line is draw, with it being either slow, medium or fast. Finally is the amount of sides to the drawing. This can be any number from 3 to 10. 3 being triangular shapes and thus the angle that the line changes direction is 60 degrees, for 4 sides, it would be 90 degrees create rectangular shapes and so on.
