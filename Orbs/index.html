Speed:
Control the rate at which the wave travels across the canvas.

Number:
Control the number of particles present in the wave.

Size:
Control the size of each particle within the wave.

Frequency:
Control the rate at which the wave oscillates

Amplitude:

Top:
Control the position of the upper limit of the wave

Bottom:
Control the position of the lower limit of the wave

Colour:
Edit the colour of each particle strand individually or randomise all colours.
